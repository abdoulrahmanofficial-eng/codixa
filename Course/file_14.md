# 🎯 أهداف الدرس

- فهم معنى إدخال البيانات (User Input).

- استقبال النصوص والأرقام من المستخدم.
- تحويل المدخلات إلى الأنواع المناسبة.

# مقدمة

حتى الآن كانت جميع برامجنا تعرض معلومات ثابتة.

مثال:

```python
print("Hello Ahmed")
```

لكن ماذا لو أردنا أن يكتب كل مستخدم اسمه بنفسه؟

أو عمره؟

أو درجته؟

هنا نستخدم **إدخال البيانات**.

---

# ما هي الدالة input()؟

الدالة `input()` تجعل البرنامج ينتظر حتى يكتب المستخدم قيمة.

بعد الضغط على Enter، يتم حفظ القيمة داخل متغير.

---

# أول مثال

```python
name = input("Enter your name: ")

print(name)
```

إذا كتب المستخدم:

```text
Ahmed
```

سيكون الناتج:

```text
Ahmed
```

---

# ماذا يحدث؟

```python
name = input("Enter your name: ")
```

تعرض الرسالة:

```
Enter your name:
```

ثم تنتظر المستخدم.

بعد إدخال القيمة والضغط على Enter، تُحفظ القيمة داخل المتغير `name`.

---

# استخدام المدخلات مع print()

```python
name = input("Enter your name: ")

print("Welcome", name)
```

إذا كتب المستخدم:

```text
Ali
```

الناتج:

```text
Welcome Ali
```

---

# ملاحظة مهمة جدًا

الدالة `input()` **تعيد دائمًا نصًا (String)** حتى لو كتب المستخدم رقمًا.

مثال:

```python
age = input("Enter your age: ")

print(type(age))
```

إذا أدخل المستخدم:

```text
20
```

الناتج:

```text
<class 'str'>
```

وليس:

```text
<class 'int'>
```

---

# لماذا هذه المعلومة مهمة؟

لأن العمليات الحسابية لن تعمل بالشكل المتوقع.

مثال:

```python
number = input("Enter a number: ")

print(number + number)
```

إذا كتب المستخدم:

```text
10
```

الناتج:

```text
1010
```

وليس:

```text
20
```

لأن البرنامج تعامل مع القيمة كنص.

---

# تحويل المدخلات إلى عدد صحيح

```python
age = int(input("Enter your age: "))

print(age)
```

الآن أصبح المتغير من النوع:

```python
int
```

---

# مثال عملي

```python
number1 = int(input("First number: "))

number2 = int(input("Second number: "))

print(number1 + number2)
```

إذا أدخل المستخدم:

```text
10

20
```

الناتج:

```text
30
```

---

# استقبال عدد عشري

```python
price = float(input("Enter price: "))

print(price)
```

إذا كتب المستخدم:

```text
19.99
```

سيتم تخزين القيمة كعدد عشري.

---

# إنشاء برنامج بسيط

```python
name = input("Name: ")

age = int(input("Age: "))

print("Hello", name)

print("Your age is", age)
```

---

# حساب سنة الميلاد

```python
current_year = 2026

age = int(input("Enter your age: "))

birth_year = current_year - age

print("Birth Year:", birth_year)
```

إذا أدخل المستخدم:

```text
20
```

الناتج:

```text
Birth Year: 2006
```

---

# حساب مساحة مستطيل

```python
length = float(input("Length: "))

width = float(input("Width: "))

area = length * width

print("Area =", area)
```

---

# دمج النصوص

```python
first_name = input("First Name: ")

last_name = input("Last Name: ")

print("Full Name:", first_name + " " + last_name)
```

# استخدام f-Strings

في Python توجد طريقة حديثة وسهلة لدمج النصوص تسمى **f-Strings**.

بدلًا من كتابة:

```python
print("Welcome", name)
```

يمكن كتابة:

```python
print(f"Welcome {name}")
```

إذا كانت قيمة `name` هي:

```text
Ahmed
```

فسيكون الناتج:

```text
Welcome Ahmed
```

يمكن استخدامها أيضًا مع أكثر من متغير:

```python
name = "Sara"
age = 18

print(f"{name} is {age} years old.")
```

الناتج:

```text
Sara is 18 years old.
```

تعتبر `f-Strings` من أفضل الطرق لعرض البيانات في Python.

# أخطاء شائعة

## 1. نسيان التحويل

❌

```python
age = input()

print(age + 5)
```

سيظهر خطأ لأن `age` نص.

الصحيح:

```python
age = int(input())

print(age + 5)
```

---

## 2. إدخال نص بدل رقم

إذا كتب المستخدم:

```text
Ahmed
```

في هذا الكود:

```python
age = int(input())
```

سيظهر خطأ لأن `"Ahmed"` لا يمكن تحويلها إلى عدد صحيح.

في الدروس المتقدمة سنتعلم كيف نتعامل مع هذه الأخطاء.

---

## 3. استخدام float بدل int أو العكس

إذا كانت القيمة قد تحتوي على كسور، استخدم:

```python
float()
```

أما إذا كانت عددًا صحيحًا فقط، فاستخدم:

```python
int()
```
