# ما هي Operators؟

الـ Operators هي رموز تستخدم لتنفيذ عمليات على البيانات.

مثل:

```
+

-

*

/

>

<

==

and

or
```

---

# أنواع Operators

في Python توجد عدة أنواع.

أهمها:

- Arithmetic Operators
- Comparison Operators
- Logical Operators
- Assignment Operators

---

# أولًا: Arithmetic Operators

تستخدم لإجراء العمليات الحسابية.

---

## الجمع

```python
print(10 + 5)
```

الناتج:

```
15
```

---

## الطرح

```python
print(10 - 5)
```

الناتج:

```
5
```

---

## الضرب

```python
print(6 * 4)
```

الناتج:

```
24
```

---

## القسمة

```python
print(20 / 4)
```

الناتج:

```
5.0
```

لاحظ أن `/` تعيد دائمًا قيمة من نوع `float`.

---

## القسمة الصحيحة

```python
print(20 // 3)
```

الناتج:

```
6
```

يتم تجاهل الجزء العشري.

---

## باقي القسمة (Modulo)

```python
print(20 % 3)
```

الناتج:

```
2
```

يستخدم كثيرًا لمعرفة:

- هل العدد زوجي؟
- هل يقبل القسمة على رقم معين؟

---

مثال:

```python
print(10 % 2)
```

الناتج:

```
0
```

إذن العدد زوجي.

---

## الأس (Power)

```python
print(2 ** 3)
```

الناتج:

```
8
```

لأن:

```
2 × 2 × 2
```

---

# مثال عملي

```python
length = 10

width = 5

area = length * width

print(area)
```

الناتج:

```
50
```

---

# ثانيًا: Comparison Operators

تعيد دائمًا:

```
True

أو

False
```

---

## يساوي

```python
print(10 == 10)
```

الناتج:

```
True
```

---

## لا يساوي

```python
print(10 != 5)
```

الناتج:

```
True
```

---

## أكبر من

```python
print(20 > 15)
```

---

## أصغر من

```python
print(5 < 2)
```

---

## أكبر من أو يساوي

```python
print(10 >= 10)
```

---

## أصغر من أو يساوي

```python
print(5 <= 8)
```

---

# جدول المقارنة

| العملية | المعنى |
|---------|---------|
| == | يساوي |
| != | لا يساوي |
| > | أكبر من |
| < | أصغر من |
| >= | أكبر من أو يساوي |
| <= | أصغر من أو يساوي |

---

# ثالثًا: Logical Operators

تستخدم لدمج أكثر من شرط.

---

## and

يعيد True إذا كانت جميع الشروط صحيحة.

```python
print(True and True)
```

الناتج:

```
True
```

---

```python
print(True and False)
```

الناتج:

```
False
```

---

## or

يكفي أن يكون شرط واحد صحيحًا.

```python
print(True or False)
```

الناتج:

```
True
```

---

## not

يعكس القيمة.

```python
print(not True)
```

الناتج:

```
False
```

---

```python
print(not False)
```

الناتج:

```
True
```

---

# مثال عملي

```python
age = 20

print(age >= 18 and age <= 30)
```

الناتج:

```
True
```

---

# رابعًا: Assignment Operators

---

الإسناد العادي

```python
x = 5
```

---

الجمع ثم الإسناد

```python
x = 5

x += 3

print(x)
```

الناتج:

```
8
```

---

الطرح ثم الإسناد

```python
x = 10

x -= 2

print(x)
```

---

الضرب ثم الإسناد

```python
x = 4

x *= 5

print(x)
```

---

القسمة ثم الإسناد

```python
x = 20

x /= 2

print(x)
```

---

# ترتيب تنفيذ العمليات

Python تنفذ العمليات بترتيب معين.

مثال:

```python
print(5 + 2 * 3)
```

الناتج:

```
11
```

لأن الضرب ينفذ أولًا.

---

إذا أردت تغيير الترتيب:

```python
print((5 + 2) * 3)
```

الناتج:

```
21
```

---

# ترتيب الأولوية

1. الأقواس `()`
2. الأس `**`
3. الضرب والقسمة
4. الجمع والطرح
5. المقارنات
6. المعاملات المنطقية

---

# مثال شامل

```python
math = 90

physics = 80

chemistry = 70

average = (math + physics + chemistry) / 3

print(average)

print(average >= 60)
```

الناتج:

```
80.0

True
```

# مشروع صغير

برنامج يحسب عمر المستخدم بعد 10 سنوات.

```python
age = int(input("Age: "))

future_age = age + 10

print(f"After 10 years you will be {future_age} years old.")
```

# مشروع آخر

حساب مساحة ومحيط مستطيل.

```python
length = float(input("Length: "))

width = float(input("Width: "))

area = length * width

perimeter = 2 * (length + width)

print(f"Area = {area}")

print(f"Perimeter = {perimeter}")
```

# أخطاء شائعة

## استخدام = بدل ==

❌

```python
print(10 = 10)
```

خطأ.

---

الصحيح:

```python
print(10 == 10)
```

---

## نسيان الأقواس

```python
5 + 2 * 3
```

قد يعطي نتيجة مختلفة عما تتوقع.

---

## استخدام and بدلاً من or

اختر المعامل المناسب حسب المطلوب.

---

# أفضل الممارسات

✅ استخدم الأقواس لتوضيح ترتيب العمليات حتى لو لم تكن ضرورية.

---

✅ لا تكتب شروطًا معقدة في سطر واحد إذا كانت تقلل من وضوح الكود.

---

✅ استخدم أسماء متغيرات واضحة بدلًا من الأرقام المباشرة (Magic Numbers).

---

# ملخص

- تستخدم Arithmetic Operators لإجراء العمليات الحسابية.
- تستخدم Comparison Operators للمقارنة بين القيم.
- تستخدم Logical Operators لدمج الشروط.
- تستخدم Assignment Operators لتحديث قيم المتغيرات.
- فهم ترتيب تنفيذ العمليات يساعد على تجنب الأخطاء.
