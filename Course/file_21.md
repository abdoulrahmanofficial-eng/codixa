# أولًا: Tuple

الـ Tuple تشبه List إلى حد كبير.

لكن يوجد فرق مهم جدًا.

**Tuple غير قابلة للتعديل (Immutable).**

أي أنه بعد إنشائها لا يمكن:

- إضافة عناصر.
- حذف عناصر.
- تعديل عنصر.

---

# إنشاء Tuple

```python
numbers = (10, 20, 30)
```

---

يمكن أيضًا حذف الأقواس في بعض الحالات:

```python
numbers = 10, 20, 30
```

لكن يفضل استخدام الأقواس لزيادة الوضوح.

---

# Tuple بعنصر واحد

```python
number = (5,)
```

لاحظ وجود الفاصلة.

بدونها:

```python
number = (5)
```

سيصبح عددًا عاديًا وليس Tuple.

---

# الوصول للعناصر

```python
colors = ("Red", "Green", "Blue")

print(colors[0])
```

الناتج:

```
Red
```

---

آخر عنصر

```python
print(colors[-1])
```

# Slicing

```python
colors = ("Red", "Green", "Blue", "Black")

print(colors[1:3])
```

الناتج:

```

```

---

# لا يمكن التعديل

❌

```python

```

سيظهر:

```

```

# متى نستخدم Tuple؟

إذا كانت البيانات ثابتة ولا يجب أن تتغير.

مثل:

- أيام الأسبوع.
- أشهر السنة.
- إحداثيات نقطة.
- أبعاد الشاشة.
- إعدادات ثابتة.

```python
days = (

    "Monday",

    "Tuesday",

    "Wednesday",

    "Thursday",

    "Friday",

    "Saturday",

    "Sunday"
```

# ثانيًا: Set

الـ Set هي مجموعة غير مرتبة من العناصر الفريدة.

أي:

- لا تسمح بالتكرار.
- لا تعتمد على الفهارس.
- لا تضمن ترتيب العناصر.

---

# إنشاء Set

```python
numbers = {1, 2, 3, 4}
```

---

إذا كررت عنصرًا:

```python
numbers = {1, 2, 2, 3, 3, 4}

print(numbers)
```

الناتج:

```
{1, 2, 3, 4}
```

يتم حذف العناصر المكررة تلقائيًا.

---

# إنشاء Set فارغة

❌

```python
data = {}
```

هذه ليست Set.

إنها Dictionary.

---

الصحيح:

```python
data = set()
```

---

# إضافة عنصر

```python
numbers.add(5)
```

---

# إضافة عدة عناصر

```python
numbers.update([6, 7, 8])
```

---

# حذف عنصر

```python
numbers.remove(5)
```

إذا لم يكن العنصر موجودًا سيظهر خطأ.

---

# discard()

```python
numbers.discard(100)
```

لن يظهر خطأ إذا لم يكن العنصر موجودًا.

---

# pop()

```python
numbers.pop()
```

يحذف عنصرًا عشوائيًا (لا تعتمد على كونه أول أو آخر عنصر).

---

# clear()

```python
numbers.clear()
```

---

# المرور على Set

```python
for number in numbers:

    print(number)
```

---

# العمليات الرياضية

نفترض:

```python
A = {1, 2, 3, 4}

B = {3, 4, 5, 6}
```

---

# Union

يجمع جميع العناصر.

```python
print(A | B)
```

أو

```python
print(A.union(B))
```

الناتج:

```
{1,2,3,4,5,6}
```

---

# Intersection

العناصر المشتركة.

```python
print(A & B)
```

أو

```python
print(A.intersection(B))
```

الناتج:

```
{3,4}
```

---

# Difference

العناصر الموجودة في A وغير الموجودة في B.

```python
print(A - B)
```

الناتج:

```
{1,2}
```

---

# Symmetric Difference

العناصر الموجودة في إحدى المجموعتين فقط.

```python
print(A ^ B)
```

الناتج:

```
{1,2,5,6}
```

---

# التحقق من وجود عنصر

```python
print(3 in A)
```

الناتج:

```
True
```

# تحويل List إلى Set

```python
numbers = [1,2,2,3,3,4]

unique = set(numbers)

print(unique)
```

الناتج:

```
{1,2,3,4}
```

---

# ثم العودة إلى List

```python
numbers = list(unique)
```

# مقارنة بين الأنواع

| الخاصية | List | Tuple | Set |
|----------|------|--------|-----|
| مرتبة | ✅ | ✅ | ❌ |
| قابلة للتعديل | ✅ | ❌ | ✅ |
| تسمح بالتكرار | ✅ | ✅ | ❌ |
| تدعم الفهارس | ✅ | ✅ | ❌ |

---

# متى أستخدم كل نوع؟

### List

عندما تتغير البيانات باستمرار.

مثل:

- المنتجات.
- الطلاب.
- المهام.

---

### Tuple

عندما تكون البيانات ثابتة.

مثل:

- الإحداثيات.
- إعدادات البرنامج.
- بيانات لا يجب تعديلها.

---

### Set

عندما تهتم بالعناصر الفريدة.

مثل:

- إزالة التكرارات.
- مقارنة مجموعتين.
- التحقق السريع من وجود عنصر.

---

# مثال عملي

إزالة التكرار من أسماء الطلاب.

```python
students = [

    "Ahmed",

    "Sara",

    "Ahmed",

    "Ali",

    "Sara"
]

unique_students = list(set(students))

print(unique_students)
```

---

# أخطاء شائعة

## استخدام الفهرسة مع Set

❌

```python
numbers[0]
```

غير مسموح.

---

## الاعتقاد أن ترتيب Set ثابت

ترتيب عناصر Set قد يختلف، لذلك لا تعتمد عليه.

---

## استخدام remove() دون التحقق من وجود العنصر

قد يسبب خطأ.

استخدم `discard()` إذا لم تكن متأكدًا من وجود العنصر.

---

# أفضل الممارسات

✅ استخدم Tuple للبيانات الثابتة.

---

✅ استخدم Set عند الحاجة إلى عناصر فريدة.

---

✅ استخدم List عندما تحتاج إلى التعديل المستمر.

---

# مشروع صغير

برنامج يجمع أسماء الطلاب ويزيل التكرار.

```python
students = []

for _ in range(5):

    name = input("Student Name: ")

    students.append(name)

unique_students = sorted(set(students))

print("\nUnique Students:")

for student in unique_students:

    print(student)
```

---

# ملخص

- Tuple تشبه List لكنها غير قابلة للتعديل.
- Set لا تسمح بالعناصر المكررة ولا تعتمد على الترتيب.
- تستخدم Set في العمليات الرياضية مثل الاتحاد والتقاطع.
- اختيار هيكل البيانات المناسب يجعل الكود أوضح وأكثر كفاءة.
