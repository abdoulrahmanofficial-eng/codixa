- استخدام المتغيرات في العمليات الحسابية.

- تجنب أشهر الأخطاء التي يقع فيها المبتدئون.

# مقدمة

في الدرس السابق تعلمنا استخدام الدالة `print()` لعرض البيانات.

لكن ماذا لو أردنا تخزين اسم المستخدم؟

أو عمره؟

أو درجته؟

هل سنكتب القيم في كل مرة؟

بالطبع لا.

لهذا نستخدم **المتغيرات**.

---

# ما هو المتغير؟

المتغير (Variable) هو مكان في ذاكرة الكمبيوتر يُستخدم لتخزين البيانات حتى يتمكن البرنامج من استخدامها لاحقًا.

فكر فيه على أنه صندوق له اسم، يمكنك وضع قيمة بداخله ثم استخدامها أو تغييرها وقتما تشاء.

---

# مثال من الحياة

تخيل أن لديك صندوقًا مكتوبًا عليه:

```
Age
```

وضعت بداخله الرقم:

```
20
```

يمكنك لاحقًا فتح الصندوق واستخدام القيمة الموجودة بداخله.

إذا تغير العمر، يمكنك استبدال القيمة بقيمة جديدة.

---

# إنشاء أول متغير

في Python:

```python
age = 20
```

في هذا السطر:

- `age` هو اسم المتغير.
- `=` تعني إسناد قيمة.
- `20` هي القيمة.

---

# عرض قيمة المتغير

```python
age = 20

print(age)
```

الناتج:

```text
20
```

---

# تخزين النصوص

```python
name = "Ahmed"

print(name)
```

الناتج:

```text
Ahmed
```

---

# تخزين الأعداد العشرية

```python
price = 99.95

print(price)
```

الناتج:

```text
99.95
```

---

# تخزين القيم المنطقية

```python
is_student = True

print(is_student)
```

الناتج:

```text
True
```

---

# تغيير قيمة المتغير

يمكن تغيير قيمة المتغير في أي وقت.

```python
score = 50

print(score)

score = 90

print(score)
```

الناتج:

```text
50
90
```

القيمة القديمة استُبدلت بالقيمة الجديدة.

---

# استخدام المتغيرات في العمليات الحسابية

```python
a = 10
b = 5

print(a + b)
```

الناتج:

```text
15
```

---

مثال آخر:

```python
length = 8
width = 4

area = length * width

print(area)
```

الناتج:

```text
32
```

---

# استخدام أكثر من متغير

```python
first_name = "Ahmed"
last_name = "Ali"

print(first_name)
print(last_name)
```

---

# دمج النصوص

```python
first_name = "Ahmed"
last_name = "Ali"

print(first_name + " " + last_name)
```

الناتج:

```text
Ahmed Ali
```

---

# قواعد تسمية المتغيرات

اسم المتغير يجب أن يكون واضحًا ويعبر عن محتواه.

أمثلة جيدة:

```python
student_name

student_age

total_price

average_score
```

---

أمثلة سيئة:

```python
a

x

abc

test123
```

هذه الأسماء لا توضح الغرض من المتغير.

---

# قواعد مهمة

✅ يبدأ الاسم بحرف أو علامة `_`.

```python
name

_age
```

---

❌ لا يبدأ برقم.

خطأ:

```python
1name
```

---

✅ يمكن أن يحتوي على أرقام بعد الحرف الأول.

```python
student1
```

---

❌ لا يحتوي على مسافات.

خطأ:

```python
student name
```

الصحيح:

```python
student_name
```

---

❌ لا تستخدم الكلمات المحجوزة في Python.

مثل:

```python
if

for

while

class
```

---

# أسلوب التسمية (Snake Case)

في Python نستخدم غالبًا:

```python
student_name

student_age

total_marks
```

يسمى هذا الأسلوب **Snake Case**.

---

# ماذا يحدث داخل الذاكرة؟

عندما تكتب:

```python
age = 20
```

يمكن تخيل الأمر بالشكل التالي:

```text
┌──────────────┐
│ age          │
├──────────────┤
│ 20           │
└──────────────┘
```

وعندما تكتب:

```python
age = 25
```

تصبح القيمة:

```text
┌──────────────┐
│ age          │
├──────────────┤
│ 25           │
└──────────────┘
```

---

# أخطاء شائعة

## 1. استخدام متغير قبل تعريفه

❌

```python
print(age)
```

سيظهر خطأ لأن `age` لم يتم إنشاؤه بعد.

الصحيح:

```python
age = 20

print(age)
```

---

## 2. استخدام اسم غير واضح

❌

```python
a = 100
```

الصحيح:

```python
student_score = 100
```

---

## 3. استخدام مسافات

❌

```python
student age = 20
```

الصحيح:

```python
student_age = 20
```

# ملخص

- المتغير هو مكان لتخزين البيانات.
- يمكن تخزين أرقام أو نصوص أو قيم منطقية.
- يمكن تغيير قيمة المتغير أثناء تنفيذ البرنامج.
- اختيار أسماء واضحة يجعل الكود أسهل في القراءة والصيانة.
