# ما هو نوع البيانات؟

نوع البيانات (Data Type) هو نوع القيمة المخزنة داخل المتغير.

على سبيل المثال:

```python
age = 20
```

القيمة هنا عدد صحيح.

بينما:

```python
name = "Ahmed"
```

القيمة هنا نص.

أما:

```python
is_student = True
```

فهذه قيمة منطقية.

---

# لماذا نهتم بأنواع البيانات؟

لأن طريقة التعامل مع البيانات تختلف حسب نوعها.

على سبيل المثال:

```python
10 + 5
```

الناتج:

```text
15
```

لكن:

```python
"10" + "5"
```

الناتج:

```text
105
```

في المثال الأول تم جمع رقمين.

أما في المثال الثاني فقد تم دمج نصين.

---

# أشهر أنواع البيانات في Python

| النوع | الاسم |
|--------|--------|
| int | عدد صحيح |
| float | عدد عشري |
| str | نص |
| bool | قيمة منطقية |

---

# أولًا: Integer (int)

يمثل الأعداد الصحيحة.

مثل:

```python
age = 20

score = 95

temperature = -5
```

كلها أعداد صحيحة.

---

# العمليات على الأعداد الصحيحة

```python
a = 10

b = 3

print(a + b)

print(a - b)

print(a * b)

print(a // b)
```

الناتج:

```text
13

7

30

3
```

لاحظ أن `//` تمثل القسمة الصحيحة.

---

# ثانيًا: Float

يمثل الأعداد العشرية.

مثال:

```python
price = 19.99

height = 175.5

pi = 3.14159
```

---

# العمليات على Float

```python
price = 50.5

tax = 14.5

print(price + tax)
```

الناتج:

```text
65.0
```

---

# الفرق بين int و float

```python
age = 20
```

نوعه:

```
int
```

---

```python
age = 20.0
```

نوعه:

```
float
```

وجود النقطة العشرية يجعل القيمة من نوع `float`.

---

# ثالثًا: String (str)

يمثل النصوص.

يكتب بين:

```python
" "

أو

' '
```

---

أمثلة:

```python
name = "Ahmed"

city = "Cairo"

language = "Python"
```

---

# دمج النصوص

```python
first = "Ahmed"

last = "Ali"

print(first + " " + last)
```

الناتج:

```text
Ahmed Ali
```

---

# تكرار النص

```python
print("Hi " * 3)
```

الناتج:

```text
Hi Hi Hi
```

---

# رابعًا: Boolean (bool)

القيم المنطقية لها احتمالان فقط.

```python
True

False
```

---

أمثلة:

```python
is_student = True

is_logged_in = False
```

---

تستخدم كثيرًا في الشروط.

مثل:

```python
if is_student:
    print("Welcome")
```

سنتعلم الجمل الشرطية بالتفصيل لاحقًا.

# معرفة نوع البيانات

تستخدم الدالة:

```python
type()
```

مثال:

```python
age = 20

print(type(age))
```

الناتج:

```text
<class 'int'>
```

---

مثال آخر:

```python
price = 99.5

print(type(price))
```

الناتج:

```text
<class 'float'>
```

---

مثال:

```python
name = "Ahmed"

print(type(name))
```

الناتج:

```text
<class 'str'>
```

---

مثال:

```python
is_admin = False

print(type(is_admin))
```

الناتج:

```text
<class 'bool'>
```

---

# تحويل أنواع البيانات

يمكن تحويل نوع البيانات باستخدام دوال مخصصة.

## تحويل إلى عدد صحيح

```python
age = int("20")

print(age)
```

---

## تحويل إلى عدد عشري

```python
price = float("15.5")

print(price)
```

---

## تحويل إلى نص

```python
score = str(100)

print(score)
```

---

## تحويل إلى قيمة منطقية

```python
print(bool(1))
```

الناتج:

```text
True
```

---

```python
print(bool(0))
```

الناتج:

```text
False
```

# ملخص

- كل متغير في Python له نوع بيانات.
- أشهر الأنواع: `int` و`float` و`str` و`bool`.
- يمكن معرفة النوع باستخدام `type()`.
- يمكن تحويل الأنواع باستخدام `int()` و`float()` و`str()` و`bool()`.
- فهم أنواع البيانات يساعد على كتابة برامج صحيحة وخالية من الأخطاء.
