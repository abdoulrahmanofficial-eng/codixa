# ما هي البرمجة؟

> مستوى الدرس: مبتدئ تمامًا
>
> مدة القراءة: 10-15 دقيقة

---

# 🎯 أهداف الدرس

بعد الانتهاء من هذا الدرس ستكون قادرًا على:

- فهم معنى البرمجة.
- معرفة لماذا نتعلم البرمجة.
- فهم كيف يفكر الكمبيوتر.
- التفرقة بين البرنامج ولغة البرمجة.
- كتابة أول سطر برمجي.

---

# ما هي البرمجة؟

البرمجة هي عملية كتابة مجموعة من التعليمات والأوامر التي ينفذها الكمبيوتر للوصول إلى نتيجة معينة.

بمعنى آخر...

أنت تخبر الكمبيوتر بما تريد أن يفعله خطوة بخطوة.

الكمبيوتر لا يستطيع التفكير أو التخمين.

هو فقط ينفذ التعليمات التي تكتبها له.

---

# مثال من الحياة

تخيل أنك تريد من شخص أن يصنع كوبًا من القهوة.

لن يكفي أن تقول:

> "اعمل قهوة."

بل يجب أن تشرح له الخطوات.

1. أحضر كوبًا.
2. ضع ملعقة قهوة.
3. أضف السكر.
4. اسكب الماء الساخن.
5. قلب جيدًا.

هذه تسمى **خوارزمية**.

والبرمجة هي كتابة هذه الخطوات ولكن بلغة يفهمها الكمبيوتر.

---

# ما هو البرنامج؟

البرنامج هو مجموعة من التعليمات المكتوبة بلغة برمجية.

أمثلة:

- واتساب
- فيسبوك
- إنستجرام
- تيك توك
- جوجل كروم
- الألعاب
- تطبيقات البنوك

كل هذه مجرد برامج.

---

# من هو المبرمج؟

المبرمج هو الشخص الذي يكتب التعليمات التي تجعل البرنامج يعمل.

فهو يحول الأفكار إلى تطبيقات يستخدمها الناس.

---

# هل الكمبيوتر ذكي؟

الإجابة:

لا.

الكمبيوتر سريع جدًا...

لكنه ليس ذكيًا.

إذا أخبرته أن يجمع رقمين فسيفعل.

إذا أخبرته أن يحذف كل الملفات فسيفعل.

هو لا يعرف إن كان الأمر صحيحًا أو خاطئًا.

---

# كيف يفهم الكمبيوتر؟

الكمبيوتر في النهاية لا يفهم العربية ولا الإنجليزية.

ولا حتى Python أو Java.

هو يفهم شيئًا واحدًا فقط:

```
0
1
```

وهذا يسمى النظام الثنائي (Binary).

ولأن كتابة ملايين الأصفار والواحدات مستحيلة للبشر...

تم اختراع لغات البرمجة.

---

# لغات البرمجة

من أشهر اللغات:

- Python
- JavaScript
- Java
- C
- C++
- C#
- Go
- Rust
- PHP
- Kotlin
- Swift

كل لغة لها استخدامات مختلفة.

---

# لماذا توجد لغات كثيرة؟

مثل اللغات البشرية.

يمكنك التحدث بالعربية أو الإنجليزية أو الفرنسية.

وفي النهاية توصل نفس الفكرة.

الأمر نفسه مع لغات البرمجة.

كلها تعطي الكمبيوتر تعليمات ولكن بطريقة مختلفة.

---

# ماذا يمكن أن تصنع بالبرمجة؟

يمكنك إنشاء:

- مواقع إلكترونية.
- تطبيقات الهاتف.
- ألعاب.
- برامج سطح المكتب.
- روبوتات.
- ذكاء اصطناعي.
- أنظمة شركات.
- تطبيقات تعليمية.
- متاجر إلكترونية.

---

# كيف يفكر المبرمج؟

أي مشكلة تمر بأربع مراحل:

```
فهم المشكلة

↓

تحليلها

↓

كتابة الحل

↓

اختبار الحل
```

---

# أول برنامج

في لغة Python:

```python
print("Hello World")
```

الناتج:

```
Hello World
```

---

# ماذا حدث؟

السطر السابق طلب من الكمبيوتر:

> اطبع كلمة Hello World على الشاشة.

وبالفعل نفذ الأمر.

---

# مثال آخر

```python
print(5 + 3)
```

الناتج:

```
8
```

---

# مثال آخر

```python
name = "Ahmed"

print(name)
```

الناتج

```
Ahmed
```

---

# معلومة مهمة

الكمبيوتر ينفذ الأوامر بالترتيب.

مثال:

```python
print("One")

print("Two")

print("Three")
```

الناتج

```
One
Two
Three
```

# أخطاء شائعة

❌ الاعتقاد أن البرمجة تعتمد على الحفظ.

الصحيح:

تعتمد على التفكير وحل المشكلات.

---

❌ الاعتقاد أن المبرمج يحفظ جميع الأكواد.

الصحيح:

حتى المحترفون يبحثون في التوثيق باستمرار.

---

❌ الاعتقاد أن تعلم البرمجة يحتاج ذكاءً خارقًا.

الصحيح:

يحتاج إلى التدريب والممارسة والاستمرارية.

---

# ملخص الدرس

- البرمجة هي كتابة تعليمات للكمبيوتر.
- الكمبيوتر لا يفكر بل ينفذ الأوامر.
- البرنامج هو مجموعة تعليمات.
- لغات البرمجة هي وسيلة للتواصل مع الكمبيوتر.
- يمكن استخدام البرمجة لبناء مواقع وتطبيقات وألعاب وأنظمة مختلفة.
