# 🎯 أهداف الدرس

بعد الانتهاء من هذا الدرس ستكون قادرًا على:

- فهم كيف يعمل الكمبيوتر.
- معرفة الفرق بين Hardware وSoftware.
- التعرف على أهم مكونات الكمبيوتر.
- فهم رحلة تنفيذ البرنامج داخل الجهاز.

---

# مقدمة

عندما تضغط على زر تشغيل الكمبيوتر، يحدث آلاف العمليات في أقل من ثانية.

قد يبدو الأمر بسيطًا، لكن خلف الكواليس يعمل عدد كبير من المكونات معًا حتى تتمكن من فتح أي برنامج أو تصفح الإنترنت.

---

# ما هو الكمبيوتر؟

الكمبيوتر هو جهاز إلكتروني يستقبل البيانات، ثم يعالجها، ثم يعرض النتائج.

ببساطة:

```
Input

↓

Processing

↓

Output
```

أي:

إدخال → معالجة → إخراج.

---

# مثال من الحياة

أنت تستخدم آلة حاسبة.

تكتب:

```
10 + 20
```

الآلة تقوم بالحساب.

ثم تعرض:

```
30
```

إذن:

Input

↓

10 + 20

↓

Processing

↓

عملية الجمع

↓

Output

↓

30

---

# المكونات الرئيسية

ينقسم الكمبيوتر إلى قسمين:

## Hardware

وهي جميع الأجزاء التي يمكنك لمسها.

مثل:

- الشاشة
- لوحة المفاتيح
- الفأرة
- المعالج
- الرامات
- الهارد

---

## Software

وهي البرامج التي تعمل على الجهاز.

مثل:

- Windows
- Google Chrome
- VS Code
- الألعاب
- Microsoft Word

---

# العلاقة بينهما

تخيل أن جسم الإنسان:

Hardware = الجسم.

Software = العقل.

الجسم بدون عقل لا يفعل شيئًا.

والعقل بدون جسم لا يستطيع الحركة.

وهكذا الكمبيوتر.

---

# المعالج (CPU)

يسمى:

Brain of the Computer

أي عقل الكمبيوتر.

هو المسؤول عن تنفيذ جميع التعليمات.

كل برنامج تقوم بفتحه يمر على المعالج.

---

# ماذا يفعل المعالج؟

يقوم بأربع عمليات:

1.

قراءة التعليمات.

↓

2.

فهمها.

↓

3.

تنفيذها.

↓

4.

إرسال النتيجة.

---

# الذاكرة RAM

RAM هي الذاكرة المؤقتة.

تخزن البرامج أثناء تشغيلها.

مثال:

عندما تفتح:

- Chrome
- VS Code
- Discord

فإنها تحفظ داخل RAM.

---

# لماذا تسمى مؤقتة؟

لأنها تمسح بالكامل بعد إغلاق الجهاز.

---

# الهارد (Storage)

هو مكان حفظ الملفات بشكل دائم.

مثل:

- الصور
- الفيديوهات
- المشاريع
- الألعاب
- نظام التشغيل

حتى بعد إطفاء الجهاز.

---

# الفرق بين RAM و Storage

| RAM | Storage |
|------|----------|
| مؤقتة | دائمة |
| أسرع | أبطأ |
| تُمسح عند الإغلاق | لا تُمسح |
| تستخدم أثناء التشغيل | تستخدم للحفظ |

---

# لوحة الأم

Motherboard

هي اللوحة التي تربط جميع القطع ببعضها.

بدونها لن يستطيع أي جزء التواصل مع الآخر.

---

# كارت الشاشة

GPU

مسؤول عن معالجة الرسوميات.

كلما كان أقوى:

- الألعاب تصبح أفضل.
- تصميم الفيديو يصبح أسرع.
- برامج ثلاثية الأبعاد تعمل بكفاءة.

---

# أجهزة الإدخال

هي الأجهزة التي تدخل البيانات.

مثل:

- Keyboard
- Mouse
- Microphone
- Camera
- Scanner

---

# أجهزة الإخراج

هي الأجهزة التي تعرض النتائج.

مثل:

- Monitor
- Printer
- Speaker
- Projector

---

# رحلة تشغيل برنامج

عندما تضغط على أيقونة Chrome يحدث التالي:

1.

النظام يبحث عن البرنامج داخل الهارد.

↓

2.

ينقل الملفات إلى RAM.

↓

3.

المعالج يبدأ تنفيذ التعليمات.

↓

4.

كارت الشاشة يرسم الواجهة.

↓

5.

تظهر نافذة البرنامج أمامك.

---

# كيف ينفذ الكمبيوتر الكود؟

عند كتابة:

```python
print("Hello")
```

يحدث الآتي:

```
الكود

↓

المفسر Interpreter

↓

لغة الآلة

↓

CPU

↓

الشاشة
```

---

# هل الكمبيوتر ينفذ أكثر من برنامج؟

نعم.

لكنه ينفذ ملايين التعليمات بسرعة كبيرة جدًا حتى يبدو وكأنه يشغل كل شيء في نفس اللحظة.

---

# معلومة

المعالج لا يفهم Python.

ولا Java.

ولا JavaScript.

كل هذه اللغات يتم تحويلها أولًا إلى Machine Code.

ثم يبدأ التنفيذ.

---

# أخطاء شائعة

❌ الاعتقاد أن الرامات تحفظ الملفات.

الصحيح:

الهارد هو الذي يحفظ الملفات.

---

❌ الاعتقاد أن المعالج يحفظ البيانات.

الصحيح:

المعالج ينفذ التعليمات فقط.

---

❌ الاعتقاد أن زيادة مساحة الهارد تجعل الجهاز أسرع.

الصحيح:

السرعة تعتمد أكثر على المعالج والرامات ونوع وحدة التخزين.

---

# ملخص

- الكمبيوتر يستقبل البيانات ثم يعالجها ثم يعرض النتائج.
- Hardware هي المكونات المادية.
- Software هي البرامج.
- CPU هو عقل الكمبيوتر.
- RAM ذاكرة مؤقتة.
- Storage للتخزين الدائم.
- جميع البرامج تمر على المعالج أثناء التنفيذ.
