# تثبيت Python

## الخطوة الأولى

## الخطوة الثانية

## الخطوة الثالثة

```

```

# التأكد من نجاح التثبيت

افتح Terminal أو Command Prompt ثم اكتب:

```bash
python --version
```

أو في بعض الأنظمة:

```bash
python3 --version
```

إذا ظهر رقم الإصدار، فهذا يعني أن Python تم تثبيتها بنجاح.

مثال:

```text
Python 3.13.2
```

قد يختلف رقم الإصدار حسب النسخة المثبتة لديك.

# ماذا يعني print؟

الدالة `print()` تستخدم لعرض البيانات على الشاشة.

مثال:

```python
print("Welcome")
```

الناتج:

```text
Welcome
```

---

# طباعة أكثر من سطر

```python
print("Hello")

print("Programming")
```

الناتج:

```text

Programming
```

كل استدعاء لـ `print()` يعرض سطرًا جديدًا.

# كتابة التعليقات (Comments)

أحيانًا تحتاج إلى كتابة ملاحظات داخل الكود.

تبدأ التعليقات بالرمز:

```python
#
```

مثال:

```python
# هذا أول برنامج لي

print("Hello")
```

التعليق لن يتم تنفيذه، وإنما يساعد في توضيح الكود.

---

# أخطاء شائعة

## 1. نسيان علامات الاقتباس

❌

```python
print(Hello)
```

سيظهر خطأ لأن Python ستعتبر Hello اسم متغير.

الصحيح:

```python
print("Hello")
```

---

## 2. نسيان القوس

❌

```python
print("Hello"
```

الصحيح:

```python
print("Hello")
```

---

## 3. حفظ الملف بامتداد خاطئ

❌

```
main.txt
```

الصحيح:

```
main.py
```
